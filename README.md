# Semicolon
**정보화 취약계층을 위한 웹 키오스크**

매장 내에 설치되어 있는 키오스크 기기에 익숙하지 않은 고객을 위해 핸드폰 카메라로 QR을 찍어 핸드폰으로 주문 가능한 웹 키오스크로, 주문에 어려움을 느끼는 고객을 위해 지인에게 주문 부탁 기능을 추가하였다.
